### Run Locally

To run **Barbeque** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/user-kartik/Barbeque.git
```

Windows:

```bash
git clone https://github.com/user-kartik/Barbeque.git
```
